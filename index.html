Pertemuan 2 MK Pemrograman Web

Pada pertemuan kedua mata kuliah pemrograman web, kami ditugaskan untuk menjelaskan jenis-jenis tag pada HTML yang sebelumnya sudah dibagikan pada pertemuan sebelumnya. Kebetulan saya mendapatkan tag HTML yang lumayan mudah yaitu tag time. Yang dimana penggunaannya itu untuk menjelaskan sebuah keterangan waktu tertentu. Tag ini juga dapat kita sertakan dengan atribut datetime untuk menerjemahkan tanggal atau waktu ke dalam format yang dapat dibaca oleh mesin. Untungnya Tag tersebut dengan mudah saya pahami, jadi semua berjalan lancar.
