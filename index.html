Pertemuan 2 MK Pemrograman Web

Pada pertemuan kedua mata kuliah pemrograman web, kami ditugaskan untuk menjelaskan jenis-jenis tag pada HTML yang sebelumnya sudah dibagikan pada pertemuan sebelumnya. Kebetulan saya mendapatkan tag HTML yang lumayan mudah yaitu tag <time>. Tag ini digunakan untuk menjelaskan keterangan waktu tertentu dan dapat disertai atribut datetime untuk format yang dapat dibaca oleh mesin. Untungnya, tag ini dengan mudah saya pahami, jadi semua berjalan lancar.
